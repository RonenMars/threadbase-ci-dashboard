# Roadmap

| Task | Summary |
|------|---------|

| [Multi-repo deployments](./tasks/multi-repo-deployments.md) | Manage `tb-streamer` (Fly.io) deploys alongside `tb-mobile`, via a projects registry, per-project dispatch forms, and project-scoped history. |
